# E-Commerce Website

A responsive E-Commerce web application built using React and Vite.

## Features

- Product Listing
- Category Filtering
- Product Search
- Product Detail Page
- Add to Cart
- Quantity Increase/Decrease
- Remove From Cart
- Total Price Calculation
- Responsive Mobile UI

## Tech Stack

- React
- Vite
- React Router DOM
- Bootstrap
- React Toastify
